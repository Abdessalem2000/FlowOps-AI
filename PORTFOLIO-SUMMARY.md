- **Frontend**: Next.js 14, React 18, TypeScript
- **Backend**: Supabase (PostgreSQL, Auth, RLS)
- **Styling**: Tailwind CSS, Dark SaaS Theme
- **Deployment**: Vercel (Production)
- **Development**: ESLint, Prettier, Git

## 📱 User Experience

### Lead Capture Flow
1. User submits form on landing page
2. Data stored in Supabase with RLS protection
3. Optional HubSpot sync (configurable)
4. Real-time dashboard update

### Dashboard Management
1. View pipeline statistics at glance
2. Search and filter leads instantly
3. Update lead status with dropdown controls
4. Add manual leads for offline sources

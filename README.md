# AI Automation Agency

B2B automation agency specializing in AI agents and workflows for lead handling, onboarding, and support.

## 🎯 AI Automation Lead Pipeline Dashboard

A full‑stack lead pipeline dashboard for agencies and e‑commerce brands. It centralizes all leads captured from public forms, shows real‑time stats (total leads, weekly leads), and lets users update lead status (New / In progress / Closed) directly from the table. Built with Next.js, Supabase (RLS), and a dark SaaS UI ready for client demos.

## 🎯 Niche
B2B agencies and SaaS companies (5-100 employees) that want to automate their core business processes using AI agents and workflows.

## 🚀 Core Automations

1. **Lead Intake & Qualification Agent** - 24/7 lead qualification and call booking
2. **Client Onboarding Workflow** - Automated client setup and project creation
3. **Support & FAQ Agent** - Instant customer support and ticket creation
